How This Works Without JavaScript:

This example uses a CSS-only approach to create a functional 3-dot menu.

HTML Structure:

  • A hidden checkbox input controls the menu visibility
  • A label styled as the 3-dot button toggles the checkbox
  • The dropdown menu is shown/hidden based on checkbox state

CSS Techniques:

  • The checkbox is hidden with display: none
  • The label is styled to look like a 3-dot button
  • The :checked pseudo-class shows/hides the menu: 
    .menu-toggle:checked ~ .menu-dropdown {
    display: block;
}
  • The ~ (general sibling combinator) selects the menu when checkbox is checked

Limitations:

  • Menu won't close when clicking outside (requires JavaScript)
  • Less flexible than JavaScript solutions
  • Still accessible as it uses standard form controls
